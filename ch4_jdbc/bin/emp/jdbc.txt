JDBC 프로그래밍 순서

1. 드라이버 로드
Class.forName("oracle.jdbc.OracleDriver"); // Oracle
Class.forName("com.mysql.cj.jdbc.Driver"); // MySQl

2. Connection 얻기
(1) Oracle
String url = "jdbc:oracle:thin:@localhost:1521:xe";
String user = "사용자 아이디";
String password = "비밀번호";

(2) MySQL
String url = "jdbc:mysql//localhost:3306/database명";
String user = "사용자 아이디";
String password = "비밀번호";

Connection con = DriverManager.getConnection(url, user, password);

3. SQL 문
(1) connection을 통해서 실행할 sql 구문 전송
con.prepareStatement("실행할 sql 구문")
    DB 내부에서는
    1 - sql 구문을 컴파일(구문 분석)
    2 - 실행 계획 세우기
(2) sql 구문 실행
    1 - select라면 executeQuery();
    2 - update, delete, insert라면 executeUpdate();
(3) 실행결과 받기
    1 - select라면 ResultSet에 담겨져 옴
    2 - update, delete, insert라면 int 값으로 넘어옴
